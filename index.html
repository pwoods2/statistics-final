What is this course about?

This course is specifically Elements of Regression, which is defined as "an analytics procedure that can generate predictions by using an easily interpreted mathematical formula." In simpler terms, linear regression is what we do to make the data easier to understand. If you have a hard time understanding the data after regression, we have missed a step somewhere. This is where the understanding of linearization (making data linear) comes to help you out.

Getting Started and Review

On your professor's site, there is decent background into what regression is and its uses in data science. This is good background, but it can be overwhelming to try to start big picture and get smaller into the minutia. When I got started with data science, I went that route, and I cannot overemphasize how much I do not recommend it. The main idea you need to start with is: "This should make my life easier." If you are going onto a tangent that is confusing you, start over. Not to say this is all roses, but a lot of times if you back yourself out of your head you can realize you are overthinking it.
